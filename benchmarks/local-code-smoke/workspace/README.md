# JS Utils Smoke Task

Implement `src/calc.js`.

The module must export:

- `add(a, b)`: numeric addition
- `multiply(a, b)`: numeric multiplication
- `median(values)`: median of a non-empty numeric array; do not mutate the input
- `slugify(input)`: lowercase URL slug, trim leading/trailing separators, collapse repeated separators

Run:

```bash
node test.js
```
